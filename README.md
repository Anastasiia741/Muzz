## Installation
**Instal project:** 
1. Clone the repository: https://github.com/Anastasiia741/Muzz
2. Navigate to the project folder: cd MuzzChat
3. Open the project in Xcode: open MuzzChat.xcodeproj
4. Press `Cmd + R` to run the application
